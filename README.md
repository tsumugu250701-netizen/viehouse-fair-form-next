# Vie house イエタテフェアアンケートLP

## Vercel公開
- Framework Preset: Next.js
- Build Command: npm run build
- Output Directory: 空欄

## Googleフォーム連携
`app/page.jsx` の `GOOGLE_FORM_ACTION` と `ENTRY` に設定済みです。

## 重要
Googleフォーム側の「プレゼント」質問の選択肢を以下3つに変更してください。
- towerカタログギフト
- 松阪牛カタログギフト
- スターバックスギフト

LP側では「Q3. 今後、モデルハウスの見学を希望されますか？」で「ぜひ見学したい」を選んだ時だけ、プレゼントを選択できます。
