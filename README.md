# Vie house イエタテフェア アンケートLP 実働フォーム版

## Vercel設定
- Framework Preset: Next.js
- Build Command: `npm run build`
- Output Directory: 空欄

## Googleフォーム連携手順
1. Googleフォームを新規作成
2. 回答タブ → スプレッドシートアイコン → 回答管理用シートを作成
3. 質問項目を以下で作成
   - お名前：記述式
   - 電話番号：記述式
   - メールアドレス：記述式
   - Vie houseを知ったきっかけは？：チェックボックス
   - 家づくりで気になっていることは？：チェックボックス
   - 今後、モデルハウスの見学を希望されますか？：ラジオボタン
   - ご希望のご連絡方法を教えてください。：ラジオボタン
   - プレゼントの応募を希望しますか？：チェックボックス
   - ご質問・ご要望など：段落
4. Googleフォームのプレビューを開き、ページソースから `formResponse` と `entry.xxxxx` を取得
5. `app/page.jsx` の `GOOGLE_FORM_ACTION` と `ENTRY` を差し替え

## 管理方法
Googleフォームの「回答」タブに自動保存されます。
スプレッドシート連携すれば、回答者一覧として管理できます。
